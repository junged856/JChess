# JChess

Developed by Edward Jung

<h1>Features:</h1>
<p>- User friendly design</p>
<p>- Board flipping</p>
<p>- Prints status of game</p>
<p>- GUI interface</p>
<p>- User can reset board whenever they wish</p>
<p>- Castling</p>
<p>- Pins</p>
<p>- En Passant</p>
<p>- Pawn promotion</p>
<p>- Checks/Checkmates</p>
<p>- Enforces legal moves</p>
