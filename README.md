# JChess

Personal programming project made by Edward Jung

Features: 
- User friendly design, board flipping, prints status of game
- GUI interface
- User can reset board whenever they wish
- Castling
- Pins
- En Passant
- Pawn promotion
- Checks/Checkmates
- Enforces legal moves
